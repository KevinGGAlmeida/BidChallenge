# Bid Challenge Automation
The purpose of this automations is to make a BID in a Webstore application

## Description

The purpose of this automations is to make a BID in a Webstore application, but, the application also sends a bid in about 0.10 seconds before the time is up


### Dependencies

* Libraries 
- selenium
- pandas
- Pylint
* Python 3.7 or latest
